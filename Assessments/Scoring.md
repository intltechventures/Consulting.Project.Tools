
# Assessment Scoring

Ideation for Scoring Range Definitions - e.g., for Gap Analysis of Capabilities, against Requirements   



- Example Scoring Definitions
  + 0 = N/A, Informational Only 
  + 1 = Gaps Exist
  + 2 = Partially Meets
  + 3 = Meets
  + 4 = Exceeds
  + 5 = Significantly Exceeds


- Alternative Scoring Definitions 
  + -3 = No plan to address gap in product roadmap
  + -2 = Gaps Exist
  + -1 = Partially Meets
  +  0 = N/A, Informational Only 
  +  1 = Meets
  +  2 = Exceeds
  +  3 = Signficantly Exceeds
