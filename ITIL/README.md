Service Management Lifecyle Stages  
1. Service Strategy  
2. Service Design  
3. Service Transition  
4. Service Operation  
5. Continual Service Improvement  

Each stage has one or more associated processes

Support each stage and process, there are:
* roles
* activities
* metrics
* procedures
* work instructions
* feedback

Process controls include:
* defined ownership
* objectives
* policies
* documentation
* feedback
